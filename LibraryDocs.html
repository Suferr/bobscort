Bobscort Library Docs

1. Set up the library

loadstring(game:HttpGet("https://bobscort.vercel.app/bobscortlibrary.lua",true))()

2. Creating a window/frame

local Frame1 = _G.Main:New("Test")

3. Creating a Button

Frame1:Button("Text Button 1", function()
    print("Hello World!")
end

4. Creating a Toggle

Frame1:Toggle("Toggle Button", false, function(state)
    print("Toggled State: " .. (state and "On" or "Off"))
end)

5. Creating a Slider

Frame1:Slider("Slider", 0, 100, 50, function(value)
    print("Slider Value: " ..value)
end)
